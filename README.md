# ShotFire

**Top-Down 2D 스텔스 액션 슈터. 적의 시야를 피해 그림자 속을 이동하고, 소음에 주의하며 은밀히 제거하세요. 4종 무기, 적 AI와의 심리전이 포인트!**

Unity C# 기반, Behavior Tree + 절차적 맵 생성.
